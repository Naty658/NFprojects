Android Trading App Using React Native and Node.js

Project Details

1. Tools Used: Developed using VS Code, React Native for the front-end, and Node.js for the back-end.

2. Language: JavaScript was used to build the app's core functionality.

3. Testing: The app was tested on both the Android Studio emulator and an actual Android device.

App Purpose

The trading app aims to provide a new way for people to exchange items instead of buying and selling them with money. Many people post items on platforms like Facebook Marketplace or eBay but don't find buyers due to financial constraints or mismatched pricing. However, they may be open to trading items they no longer need and have been sitting unused at home. This creates a mutual exchange where both sellers and buyers can get what they want without spending money.
