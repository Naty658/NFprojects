QT Creator Buffer Overflow Detection Tool

Project Details

1. Purpose: Detects buffer overflows in C++ and C code statically, without running the code.

2. Technology: Uses a regular expression algorithm with defined rules to identify buffer overflow vulnerabilities in code.

3. Static Detection: The tool analyzes code without executing it, identifying potential issues based on predefined patterns.

How It Was Done

1. GUI Interface: Developed using Qt Creator to create a user-friendly interface for interacting with the tool.

2. Research and API Detection: Researched common APIs that could cause buffer overflow issues and created rules to detect their presence in the code.
